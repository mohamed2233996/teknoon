## مفهوم العوامل في جافاسكربت

العوامل ( operators ) عبارة عن رموز لها معنى محدد و يمكننا تقسيمها إلى 6 مجموعات أساسية كما يلي:

*   Arithmetic Operators
*   Comparison Operators
*   Logical Operators
*   Bitwise Operators
*   Assignment Operators
*   Type Operators
