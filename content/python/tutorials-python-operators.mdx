## مفهوم العوامل

العوامل ( operators ) عبارة عن رموز لها معنى محدد و يمكننا تقسيمها إلى 7 مجموعات أساسية هي:

*   Arithmetic Operators
*   Comparison Operators
*   Logical Operators
*   Bitwise Operators
*   Assignment Operators
*   Membership Operators
*   Identity Operators
