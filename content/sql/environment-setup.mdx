## كيف يتنفذ الإستعلام الذي يتم إرساله لقاعدة البيانات

أي أمر SQL يتم إرساله إلى قاعدة البيانات يسمى إستعلام ( Query ).  
الإستعلام لا يتنفذ بشكل مباشر بل يمر بعد مراحل كما يلي حتى يتنفذ و من ثم يرجع النتيجة.

![كيف يتنفذ الإستعلام في قاعدة البيانات](/content-images/sql/environment-setup/1784809625405-gninnc.png)

عند محاولة تنفيذ أي إستعلام فإن خادم قاعدة البيانات ( Database Server ) يقوم بتفسير الأوامر الموضوعة فيه. إذا كانت أوامر الإستعلام مكتوبة بشكل صحيح فإنه يقوم بتنفيذها و من ثم يرجع النتيجة.

في حال كان الإستعلام مكتوب بشكل خاطئ أو حدثت أي مشكلة تسببت في عدم تنفيذه فإن خادم قاعدة البيانات سيعطيك رسالة الخطأ و التي من خلالها يمكنك معرفة سببه و تصحيحها.

## تجهيز بيئة عمل لتعلم قواعد البيانات

هناك برامج عديدة يمكن استخدامها للتعامل مع قواعد البيانات، منها ما هو مجاني و منها ما هو مدفوع.  
في هذه الدورة سنتعامل مع قواعد بيانات MySQL لأنها مجانية و تعتبر من أشهر قواعد البيانات.

في هرمش نستخدم قواعد بيانات MySQL كما ستتعلم أنت.
