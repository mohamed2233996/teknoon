## مفهوم الرابط

الرابط ( Link ) عبارة عن نص يشير لعنوان صفحة أخرى عند النقر عليه يقوم المتصفح بنقلك إليها.

يمكن أن يكون الرابط يشير لملف أو برنامج، عندها بمجرد النقر عليه يسألك المتصفح ما إن كنت تريد تحميله أم لا.

  

### أشكال الروابط

الروابط يمكن وضعها في الصفحة بأشكل مختلفة كما يلي:

*   يمكن وضع الرابط كنص ( Text Link ).
*   يمكن وضع الرابط كزر ( Button Link ).
*   يمكن وضع الرابط كصورة ( Image Link ).

في هذا الدرس ستتعرف على الوسم المخصص لإضافة روابط بالإضافة إلى كافة الخصائص التي يمكن وضعها فيها.
