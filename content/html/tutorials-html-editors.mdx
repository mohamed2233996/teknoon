## مفهوم المحرر

المحرر ( Editor ) هو أي برنامج يمكن استخدامه للتعديل على محتوى الملفات النصية.

أنظمة التشغيل عادةً ما يكون فيها محرر نصوص ( Text Editor ) جاهز، فمثلاً في نظام ويندوز يوجد محرر جاهز إسمه Notepad، و في نظام ماك يوجد محرر جاهز إسمه TextEdit، و في نظام لينكس يوجد محرر جاهز إسمه Gedit أو Leafpad أو غيرهما على حسب التوزيعة التي تستخدمها.

المحررات الإفتراضية التي تأتي مع أنظمة التشغيل ليست أفضل محررات يمكن استخدامها لكتابة كود HTML و CSS و Javascript لأنها لا تساعدك في كتابة الكود و تصحيح الأخطاء الموجودة فيه و لا تظهر لك ألوان جميلة كما تفعل باقي المحررات المخصصة لذلك.

في هذا الدرس ستتعلم كيفية كتابة الكود بواسطة برنامج VSCode و كيفية تجربته بشكل مباشر في [محرر الويب](https://harmash.com/tryit) الخاص بموقع هرمش.
