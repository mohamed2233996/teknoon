## مفهوم المتغيرات

المتغير ( Variable ) عبارة عن مكان يتم حجزه في الذاكرة ( RAM ) بهدف تخزين قيمة معينة فيها أثناء تشغيل البرنامج.  
القيمة التي نضعها في المتغير تعتبر قيمة مؤقتة حيث أنه يتم حذفها من الذاكرة بمجرد أن يتوقف البرنامج.
