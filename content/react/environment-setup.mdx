## متطلبات العمل بواسطة React

للبدء باستخدام React يجب تنصيب Nodejs على الحاسوب لأنه من خلاله يمكن تحميل المكتبات اللازمة لإنشاء المشروع، إضافة مكتبات خارجية، إنشاء مكونات، بناء المشروع، استضافته و تصديره.

Nodejs عبارة عن خادم ( Server ) مبني بلغة جافاسكربت و هو يتضمن الأدوات الازمة التي يمكن من خلالها إنشاء و استضافة مشاريع الويب، بالإضافة إلى تحميل المكتبات الخارجية و إضافتها في المشروع بشكل تلقائي.

[موقع تحميل Nodejs](https://nodejs.org/en/download/prebuilt-installer)

  

### التأكد من وجود Nodejs

إفتح موجه الأوامر و أكتب الأمر `node -v` لأنه إن كان مثبتاً بشكل صحيح فإنه سيعرض لك رقم إصدار النسخة التي تم تنصيبها كما يلي.

#### Command Prompt

**\>** node -v  
v20.15.1

في حال كان موجه الأوامر مفتوحاً من قبل أن يتم تنصيب Nodejs فإن الأمر `node -v` لن يعمل فيه. لحل هذه المشكلة، قم فقط بفتح موجه أوامر جديد و كتابة الأمر فيه و عندها سيعمل بلا أي مشاكل.

  

### الأدوات المرفقة مع Nodejs

*   NPM و نستخدمها لتحميل المكتبات الموجودة على شبكة CDN.
*   NPX و نستخدمها لتنفيذ أوامر موجودة في المكتبات التي سبق و تم تحميلها.

NPM هي اختصار لجملة Node Package Manager و NPX هي اختصار لجملة Node Package Executor.
