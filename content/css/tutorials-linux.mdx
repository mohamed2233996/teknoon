## ما هو لينكس؟

لينكس ( Linux ) هو نظام تشغيل ( Operating System ) مفتوح المصدر ( Open Source )، يشبه إلى حد كبير أنظمة التشغيل الشائعة الأخرى مثل ويندوز ( Windows ) و ماك ( Mac ) و أندرويد ( Android ).

مفتوح المصدر، تعني أنه مجاني و يستطيع أي أحد رؤية أوامره و التعديل عليها بشكل مباشر.
