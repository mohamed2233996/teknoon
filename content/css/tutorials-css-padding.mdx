## مفهوم الهامش الداخلي

الهامش الداخلي ( Padding ) هو المسافة الفارغة التي يمكن وضعها بين محتوى العنصر و حدوده.

حجم الهامش الداخلي للعنصر يمكنك تحديده من كل الجهات أو من جهات محددة.
